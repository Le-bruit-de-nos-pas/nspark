Overview
Engine to generate HTML QC reports from two inputs:
  • QC Summary (Cohort/variable-level metrics)
  • QC Audit (Patient-level issues)
It transforms raw .csv from qc_pipeline.py into visually structured reports.
Cohort-level Summary Metrics

total_vars = len(summary)
total_issues = len(audit)

missing_vars = (summary["missing_pct"] > 0).sum()
out_range_vars = (summary["out_of_range_pct"] > 0).sum()
High-level overview of dataset quality:
  • Total variables checked
  • Total issues detected
  • Variables with missing values
  • Variables with out-of-range values
PASS / FAIL Classification

summary["status"] = summary.apply(
    lambda r: "FAIL" if (r["missing_pct"] > 0 or r["out_of_range_pct"] > 0) else "PASS",
    axis=1
)
Each variable is classified:
  • PASS → no missing or out-of-range values
  • FAIL → at least one issue detected
Any deviation triggers a failure flag.
Row Highlighting

def highlight_summary(row):
    if row["status"] == "FAIL":
        return ['background-color: #2d1111'] * len(row)
    return [''] * len(row)
Any row with issues is visually flagged (conservative).
